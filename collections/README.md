# Collections

Здесь можно хранить будущие наборы правил под конкретные окружения или типы проектов.

Примеры возможных коллекций:

- `personal/` - личные правила и привычки.
- `team/` - командные стандарты.
- `production/` - строгие правила для production-сервисов.
- `infra/` - наборы для Ansible и observability.

Пока каталог оставлен как точка расширения.
